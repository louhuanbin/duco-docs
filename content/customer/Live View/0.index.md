## Key Features

Live View is designed to give operators real-time visibility into machine status, location, and performance:

- **Real-Time Machine Overview**: Instantly access a complete list of machines with live positioning on the map.
- **Detailed Base Information**: Centralized access to machine info, tags, and ECU mapping data.
- **Alert & DTC Tracking**: Stay informed with real-time alerts and historical alarm records.
- **Route & Trace Visualization**: Review machine movement history through interactive trace playback.
- **Live Dashboard Monitoring**: Mirror on-device dashboard data for instant operational insight.
- **Battery Status Analysis**: Monitor battery health with SOC and SOH metrics.
- **High-Frequency Live Data**: Stream and visualize real-time telemetry data with interactive charts.
