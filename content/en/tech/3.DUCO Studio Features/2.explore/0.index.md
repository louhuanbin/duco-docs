## Adoption

The data explore module is powered by Grafana.

- Raw data explore is powered by grafana repeatable viz panels
- Alerting module is directly using grafana alerting module

## Technical setup

- User accessing data explore service
- Backend api will automatically detect user's org
  - If user's org not exist, the org/user/service acount will be created
  - If user's org exist, service with right contents
